Python app prompts the user to think of a number between 1 and 10 and then the computer repeatedly guesses the number until it guesses the correct number.
Can be run from the command line using docker run -i -t number_guesser. -i and -t are required for user input to work properly.
File labeled only number_guesser contains the Dockerfile
